# Introduction

API REST de Fakunet para gestionar productos, marcas, categorías, mensajes y slides.

<aside>
    <strong>Base URL</strong>: <code>http://127.0.0.1:8000</code>
</aside>

    Bienvenido a la documentación de la API de Fakunet.

    Autenticación: usa el encabezado <b>Authorization: Bearer &lt;token&gt;</b> para acceder a los endpoints protegidos.
    En la sección "Try it out" puedes probar las rutas directamente desde el navegador.

    <aside>Ejemplos de código están disponibles y puedes cambiar el lenguaje con las pestañas superiores.</aside>
